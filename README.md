# 📄 Smart Resume Screening AI

This project is an AI-based Resume Screening System that ranks resumes based on job description using NLP techniques.

## 🚀 Features
- Upload PDF resumes
- Extract text automatically
- Rank candidates based on job description
- Display match scores with visualization

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLP (TF-IDF, Cosine Similarity)

## ▶️ How to Run

1. Install dependencies:
pip install streamlit pandas scikit-learn PyPDF2

2. Run the app:
streamlit run app.py
